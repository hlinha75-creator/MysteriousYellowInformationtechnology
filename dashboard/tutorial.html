Guia rapido da guild

Aprenda a usar o bot sem depender de tentativa e erro.

Um tutorial pratico para membros registrarem nick, consultarem saldo e participarem dos eventos; e para staff administrar paineis, relatorios, auditorias e importacoes.

Sou membro Sou staff
1 Use comandos com /

No Discord, digite a barra e escolha o comando do Notag Bot na lista.

2 Leia as opcoes

Alguns comandos pedem arquivo, codigo, membro, data ou imagem.

3 Confirme antes de aplicar

Fluxos sensiveis, como importacao e verificacao, usam confirmacao.

Membros

O que cada membro precisa saber

Registrar nick

Use /registro para abrir o fluxo de registro do nome em jogo. Informe o nick correto do Albion para facilitar auditorias e organizacao de eventos.

Consultar saldo

Use /saldo para consultar seu saldo. Se precisar ver outra pessoa, selecione o membro na opcao do comando, quando permitido.

Entrar em eventos

Acompanhe os paineis fixos e comandos de evento. Quando houver codigo de evento, use /evento com o codigo informado pela staff.

Participar de enquetes e leiloes

Vote nas enquetes do canal de eventos e acompanhe leiloes publicados pelo bot. Leia prazos, valores e imagens antes de participar.

Staff

Rotina segura para operar o bot

Checklist de operacao

Setup

Paineis fixos

Use /setup para postar ou atualizar os paineis do bot. Esse comando exige permissao de gerenciamento do servidor.

Dados

Importar e exportar CSV

Use /importar para saldos e /exportar para gerar CSV de saldos, financeiro, auditoria ou voz.

Auditoria

Validar guild e canais

Use /verificar_guild, /aplicar_verificacao_guild, /auditar_canais e /renomear_canais com cuidado.

Relatorios

Fechamento diario

Use /relatorio_diario com arquivo atual, arquivo anterior e CSV de voz para comparar membros e atividade.

Referencia

Comandos do bot

Duvidas comuns

Antes de chamar a staff

O comando nao apareceu no Discord.

Confira se voce esta no servidor correto e se o bot esta online. Comandos novos podem exigir registro com npm run deploy:commands pela administracao tecnica.

Meu nick esta errado.

Use /registro novamente ou fale com a staff para revisar seu cadastro.

Qual formato de data usar?

Quando o comando pedir data, use AAAA-MM-DD, por exemplo 2026-06-12.

Posso importar qualquer CSV?

Nao. Use apenas arquivos esperados pelo fluxo do bot e revise o resumo antes de confirmar importacoes.
